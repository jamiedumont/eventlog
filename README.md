# EventLog
=======

EventLog is an indexed append log written in Elixir. 

## Quickstart

* add event_log to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:event_log, "~> 0.1.0"}
  ]
end
```

* run:

    ```
    mix deps.get
    ```
